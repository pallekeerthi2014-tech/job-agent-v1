# Basic SEO Checklist

- Confirm the page `<title>` clearly names The Kutiram and the farm stay offering
- Confirm the meta description is present and aligned with the current positioning
- Add a production canonical URL once the final domain is chosen
- Add Open Graph tags for title, description, URL, and preview image
- Add Twitter card tags if social sharing matters
- Ensure every visible image has meaningful `alt` text
- Generate `robots.txt` and `sitemap.xml` once the final public URL is known
- Compress large photos before final launch for better Core Web Vitals
- Test mobile performance and image lazy loading
- Run Lighthouse for performance, SEO, accessibility, and best practices
- Submit the live domain to Google Search Console after launch
- Recheck headings so there is one clear `h1` and logical section hierarchy
